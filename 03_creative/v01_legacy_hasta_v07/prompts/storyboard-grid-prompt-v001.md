# Storyboard Grid Prompt v001

## INSTRUCCION PRINCIPAL

Crear una sola imagen a modo de storyboard / thumbnail board con las 34 tomas descritas abajo.

La imagen debe funcionar como una hoja visual de referencia cinematografica: 34 thumbnails limpios en formato 16:9, organizados en una cuadricula/contact sheet ordenada.

Muy importante: la imagen final NO debe incluir numeracion de tomas, nombres de escenas, texto descriptivo, captions, labels, subtitulos, bordes, separadores graficos, UI de storyboard, marcas de agua ni ningun texto visible. Solo imagenes cinematograficas limpias.

Cada thumbnail debe representar visualmente una toma distinta, respetando el orden de lectura de izquierda a derecha y de arriba hacia abajo.

## USO PREVISTO

Este documento funciona como prompt maestro / biblia visual completa del storyboard.

Puede usarse para generar una hoja general de referencia, pero su valor principal es servir como base para derivar despues:

* Keyframes individuales por toma.
* Prompts de movimiento por bloque o por toma.
* Estructura de nodos en Krea.
* Referencias para foto-video montaje.
* Capa posterior de UI real sobre pantallas placeholder.

## DIRECCION DE ARTE / LOOK & FEEL

* Tono fotografico suave, sin HDR agresivo.
* Negros ligeramente levantados, sin negros aplastados.
* Altas luces con roll-off suave tipo cine analogico.
* Grano fino cinematografico.
* Mas cine autoral que publicidad comercial.
* Menos contraste, mas atmosfera.
* Mas silencio visual.
* Mas textura que nitidez digital.
* Luz natural primero, luz practica despues.
* Sensacion ligeramente nostalgica pero contemporanea.

Color:

* Verdes naturales, no saturados.
* Amarillos calidos y acentos suaves de golden hour.
* Sombras con leve tendencia fria.
* Verde oscuro y dorado calido solo para momentos de superficies digitales abstractas tipo T1.
* Uniforme de jugadores: playera verde con dos lineas blancas como acento grafico original, short blanco y calcetas rojas color #DA3B2B.
* Uniforme de equipo contrario: playera blanca, short azul y calcetas blancas, sin logos ni escudos.
* Playera/jersey color #DA3B2B para protagonistas de comercio/restaurante cuando se busque contraste visual.
* Uniforme de arbitros: playera verde fosforescente, short negro, sin logos ni insignias.

Texturas visibles:

* Metal.
* Piel.
* Tela.
* Carton.
* Fruta.
* Concreto.
* Cesped.
* Polvo y haze.
* Jerseys con textura sutil geometrica abstracta, inspirada libremente en patrones prehispanicos, sin copiar ningun simbolo especifico.

Iluminacion:

* Luz natural y practica.
* Contraluces suaves.
* Haze ligero en escenas deportivas.
* Nada de sobredrama artificial.
* Nada de iluminacion comercial stock.

Lente / camara:

* Sensacion dominante de 35mm / 50mm.
* Profundidad de campo media a shallow.
* Stills cinematograficos 16:9.
* Composiciones limpias y legibles.
* Ningun texto visible dentro de ninguna imagen.

Pantallas / dispositivos:

* Todas las pantallas de smartphone, smartwatch, tablet o laptop deben funcionar como superficies placeholder para composicion en postproduccion.
* No generar UX especifica, no generar interfaz propietaria, no generar texto legible, no generar numeros, no generar iconos de marcas, no generar codigos, no generar botones con labels.
* Las pantallas pueden mostrar bloques abstractos, campos vacios, tarjetas genericas, lineas tacticas o formas suaves de color que sugieran estructura, siempre sin informacion legible.
* La intencion es reservar esas superficies para foto-video montaje posterior.
