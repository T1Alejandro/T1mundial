# Visual Reference Fit Review

Revision de las referencias en `02_references/visual` contra el estilo definido para el storyboard: cine autoral suave, negros levantados, textura real, luz natural/practica, menos contraste, atmosfera, comercio mexicano y futbol ficticio sin marcas.

## Resumen Ejecutivo

Las referencias si sirven, pero no todas como referencia directa.

* Mejores referencias de estilo general: `1.55.53 p.m.`, `1.58.22 p.m.`, `2.06.40 p.m.`
* Mejores referencias para cierre abstracto: `1.50.37 p.m.`
* Referencias utiles solo parcialmente: `1.47.30 p.m.`, `2.10.12 p.m.`
* Referencia a evitar como look principal: `2.12.46 p.m.`

## Evaluacion Por Archivo

### Captura 1.47.30 p.m.

Fit: parcial.

Que si tomar:
* Luz natural suave.
* Grano / textura organica.
* Silencio visual.
* Composicion con mucho espacio negativo.
* Sensacion introspectiva y no publicitaria.

Que no tomar:
* Contexto belico / europeo.
* Vestuario militar.
* Fisionomia no alineada con casting mexicano / latinoamericano.
* Paleta demasiado campo europeo si se vuelve literal.

Uso recomendado:
Referencia de tono y textura, no de contenido.

### Captura 1.50.37 p.m.

Fit: alto para tomas 33-34.

Que si tomar:
* Espacio negativo.
* Sobriedad.
* Escala cosmica.
* Tierra desplazada al borde del frame.
* Silencio visual premium.
* Look espacial limpio, sobrio y sin informacion grafica.

Que ajustar:
* Menos azul frio brillante.
* Mexico debe sentirse como resplandor tenue, no como imagen satelital informativa.
* Evitar look documental espacial demasiado limpio.

Uso recomendado:
Referencia principal para cierre cosmico / toma 34.

### Captura 1.55.53 p.m.

Fit: alto.

Que si tomar:
* Interior real, vivido, cotidiano.
* Luz natural lateral.
* Profundidad por capas.
* Dignidad domestica / humana.
* Color contenido con acentos calidos.
* Sensacion de espacio mexicano sin convertirlo en postal.

Que no tomar:
* No hacer la escena demasiado domestica si la toma pide comercio o emprendimiento.
* Evitar elementos reconocibles o textos.

Uso recomendado:
Referencia fuerte para comercio, emprendedores, restaurante y capas de profundidad.

### Captura 1.58.22 p.m.

Fit: alto, con cautela.

Que si tomar:
* Centro historico mexicano.
* Haze urbano.
* Profundidad de calle.
* Dignidad cotidiana.
* Color suave, no hipercontrastado.
* Sensacion de ciudad viva.

Que no tomar:
* Banderas visibles.
* Texto legible en puesto.
* Arquitectura demasiado reconocible o landmark.
* Cualquier elemento patriotico literal.

Uso recomendado:
Referencia fuerte para tomas urbanas / salida a superficie / entorno CDMX, limpiando texto y banderas.

### Captura 2.06.40 p.m.

Fit: alto para camara y energia urbana.

Que si tomar:
* Camara baja.
* Movimiento cotidiano.
* Capas de autos / cuerpos / calle.
* Sensacion documental.
* Realismo de ciudad.

Que no tomar:
* Caos excesivo.
* Distracciones que roben protagonismo.
* Animales o elementos accidentales que cambien la lectura.
* Paleta demasiado dura de mediodia.

Uso recomendado:
Referencia para angulos bajos, movimiento de piernas y energia de calle; util para paralelos con futbol.

### Captura 2.10.12 p.m.

Fit: parcial.

Que si tomar:
* Haze deportivo.
* Espacio de cancha.
* Luz practica.
* Atmosfera suspendida.

Que no tomar:
* Azul/teal demasiado dominante.
* Look de cancha vacia demasiado artificial.
* Exceso de dramatizacion de luces.
* Sensacion mas comercial / videoclip si se copia literal.

Uso recomendado:
Referencia secundaria para tunel/cancha/haze, bajando saturacion teal y dramatismo.

### Captura 2.12.46 p.m.

Fit: bajo para este proyecto.

Que si podria tomar:
* Uso de espacio negativo.
* Silueta deportiva aislada.

Que no tomar:
* Uniforme real / nacional.
* Logos visibles.
* Look publicitario.
* Contraste alto.
* Foco en celebridad/atleta reconocible.
* Puesta de luz demasiado comercial.

Uso recomendado:
No usar como referencia principal. Si acaso, solo como recordatorio de lo que NO queremos: no equipos reales, no kits oficiales, no logos, no publicidad deportiva.

## Nota Para Prompts

Cuando se usen estas referencias con ChatGPT Images o Krea, conviene decir:

* Usar como referencia de luz, textura, atmosfera y composicion.
* No copiar contenido literal.
* No copiar logos, textos, banderas, uniformes reales ni landmarks.
* Mantener casting mexicano / latinoamericano para comercio, ciudad, protagonistas, espectadores y equipo principal.
* Arbitros y equipo contrario blanco/azul pueden ser neutrales / internacionales.
