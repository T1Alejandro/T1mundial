# Assets README

Esta carpeta guarda assets aprobados o en proceso. Los archivos pesados pueden vivir en Drive/Adobe/Frame.io, pero aquí debe quedar un índice o placeholder con link.

## Subcarpetas

- `brand/`: logos, tokens, lineamientos.
- `characters/`: cast AI, referencias, fichas de continuidad.
- `locations/`: tiendas, mercado, bodega, estadio, México aéreo.
- `ui/`: pantallas propias T1 para dashboards, pagos, envíos, tiendas.
- `logos/`: versiones oficiales y usos aprobados.
- `fonts/`: tipografías autorizadas o links de licencia.

## Regla

Nada entra a `final` sin dueño, fecha, fuente, derechos y estado.
