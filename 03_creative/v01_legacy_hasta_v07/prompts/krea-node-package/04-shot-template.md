# Shot Prompt Template

Usar esta plantilla para convertir cada toma del storyboard maestro en keyframe individual.

## Toma XX

Shot intent:
[Una frase con la funcion narrativa de la toma.]

Image prompt:
[Descripcion visual limpia de la toma. Mantenerla concreta. No meter toda la biblia global aqui.]

Apply shared blocks:

* Global Style.
* Camera / Lens.
* Color Continuity.
* Texture Continuity, si aplica.
* Wardrobe Continuity, si aplica.
* Generic Digital Screen Treatment, si hay pantallas.
* Global Negative Prompt.

Output:

* Single cinematic keyframe.
* 16:9.
* No text.
* No labels.
* No logos.

Review checklist:

* Se entiende la accion principal.
* No aparece texto accidental.
* No aparecen logos o marcas.
* Pantallas digitales genericas y abstractas sin informacion legible.
* Uniformes ficticios, no oficiales.
* Look suave, no stock, no HDR.
