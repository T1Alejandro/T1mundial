# Character Model Sheet Visual System — v008 v001

## Nota Interna

No pegar esta sección en Higgsfield. Sirve solo para ordenar el flujo de producción.

El objetivo es crear fichas visuales individuales para cada personaje principal de T1 v08 antes de producir keyframes. Estas fichas fijan casting, identidad, vestuario, ángulos, props, paleta, textura, locación y prompts base.

Dirección clave para nosotros:

- 90% imagen.
- 10% texto máximo.
- Priorizar 360 del personaje, cuerpo completo, rostro en ángulos, detalles de vestuario, manos, props y entorno.
- El texto dentro de la imagen debe ser mínimo, funcional y secundario.
- Los prompts completos van en este documento, fuera de la imagen generada.
