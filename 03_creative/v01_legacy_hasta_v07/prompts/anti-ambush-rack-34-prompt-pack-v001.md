# Anti-Ambush Rack 34 Prompt Pack

## Uso Rapido

Para cada still:

1. Pegar el `Prompt Maestro`.
2. Pegar la toma correspondiente.
3. Generar imagen 16:9.
4. Si la imagen mete estadio, porteria, jersey, logos, TV deportiva o crowd, regenerar.

## Prompt Maestro

```text
Genera UNA imagen cinematografica 16:9 como still de pelicula para una campana de comercio mexicano. Realismo curado, fotografia suave, grano fino, roll-off analogico, contraste bajo-medio, luz natural/practica, textura real de piel, tela, metal, carton, polvo y ciudad. Paleta: verdes cotidianos poco saturados, azules escolares, amarillos de atardecer, sombras frias suaves, acento rojo T1 #DB3B2B solo en objetos pequenos o marca propia cuando se pida.

La historia cruza comercio, familia, escuela, juego infantil, preparacion, velocidad y celebracion cotidiana. Debe sentirse grande y emocional, pero nacida de algo pequeno y humano. Nada debe parecer torneo profesional, patrocinio oficial ni evento masivo.

Casting: personas mexicanas/latinoamericanas reales, ninos y adultos naturales, no modelos publicitarios perfectos, no celebridades, no atletas reales. Vestuario: ropa escolar generica, streetwear sencillo, mandiles, ropa de negocio, tenis genericos sin logos, nada de uniformes deportivos oficiales ni jerseys.

NEGATIVOS OBLIGATORIOS:
No estadio profesional. No porteria profesional. No cancha profesional. No torneo. No evento masivo. No patrocinio oficial. No FIFA. No World Cup. No Copa del Mundo. No Mundial. No Mexico 2026. No seleccion. No uniforme deportivo oficial. No jersey nacional. No escudos. No logos. No marcas deportivas. No Nike. No Adidas. No Puma. No arbitro profesional. No entrenador profesional. No portero profesional. No guantes profesionales. No balon profesional. No red profesional. No aficion de estadio. No crowd masivo. No transmision deportiva. No trofeo. No mascota. No bandera como identidad deportiva. No canticos. No frases "vamos Mexico", "si se puede" ni narracion deportiva. No texto ilegible. No marcas de pago. No QR. No UI de terceros. No TV mostrando partido. No pantalla reconocible. No landmarks reconocibles. No stock look. No HDR. No manos deformes. No dedos extra. No caras deformes.
```

## Prompt De Motion Global

```text
Use the provided still as strict visual anchor. Animate with restrained cinematic motion only: slow push-in, subtle handheld, shallow rack focus, atmospheric dust, cloth movement, natural breathing, small object motion. Preserve identity, framing, wardrobe, props and environment. Do not invent logos, text, stadium, sports uniforms, professional goal, crowd, broadcast elements or official event cues. Keep motion grounded, human and physically plausible.
```
