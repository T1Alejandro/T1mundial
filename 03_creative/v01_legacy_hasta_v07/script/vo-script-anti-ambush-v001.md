# VO Script Anti-Ambush v001

## Objetivo

Mantener la cadencia emocional del guion original, pero reducir la lectura explicita de partido profesional, evento masivo o patrocinio oficial.

## Version Recomendada

```text
Hay dias que no son como los demas.

Dias donde uno se prepara
para algo grande.

Donde cada movimiento cuenta.

Donde no se falla
en los detalles.

Donde el equipo es todo.

Hay quien lo entrena
toda la vida.

Hay quien lo entrena
cada manana.

Hay quien estudia
al de enfrente.

Hay quien estudia
el mercado.

Hay quien sabe que todo
se mueve rapido.

Y hay quien sabe
que el dinero tambien.

Hoy todo Mexico
se mueve.

Cada quien
en lo suyo.

Millones
en las suyas.

Mismo impulso.

Mismo pais.

Todo Mexico es uno.

T1.

Todo en uno.
```

## Lineas Que Se Sustituyen

| Original | Sustitucion recomendada | Motivo |
|---|---|---|
| Hay quien estudia al rival. | Hay quien estudia al de enfrente. | Mantiene tension sin lenguaje deportivo directo. |
| Hay quien sabe que el balon se mueve rapido. | Hay quien sabe que todo se mueve rapido. | Quita balon profesional/evento. |
| Hoy todo Mexico juega. | Hoy todo Mexico se mueve. | Evita lectura de evento deportivo nacional. |
| Once en la cancha. | Cada quien en lo suyo. | Quita referencia a equipo/futbol profesional. |
| Millones en las suyas. | Millones en las suyas. | Se conserva; funciona para negocios/personas. |
| Mismo equipo. | Mismo impulso. | Evita asociacion deportiva directa. |
| Mismo pais. | Mismo pais. | Se conserva; amplio y no oficial. |

## Version Mas Cercana Al Original

```text
Hay dias que no son como los demas.
Dias donde uno se prepara para algo grande.

Donde cada movimiento cuenta.
Donde no se falla en los detalles.
Donde el equipo es todo.

Hay quien lo entrena toda la vida.
Hay quien lo entrena cada manana.

Hay quien estudia al de enfrente.
Hay quien estudia el mercado.

Hay quien sabe que todo se mueve rapido.
Y hay quien sabe que el dinero tambien.

Hoy todo Mexico se mueve.

Cada quien en lo suyo.
Millones en las suyas.

Mismo impulso.
Mismo pais.

Todo Mexico es uno.
T1.
Todo en uno.
```

## Notas Legales / Creativas

- Evitar "juega", "cancha", "balon", "rival" y "once" en VO final si la imagen ya contiene juego escolar.
- El tagline final queda como excepcion de negocio: "Todo Mexico es uno. T1. Todo en uno."
- Si legal pide una version mas conservadora, cambiar "Hoy todo Mexico se mueve" por "Hoy todo se mueve".
