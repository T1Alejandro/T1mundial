# Sprint 0 Plan - Setup and First Proof

## Objetivo

Dejar el proyecto operativo y probar el riesgo más grande: si podemos lograr una pieza cinematográfica con AI, legalmente segura y emocionalmente clara.

## Duración sugerida

28 de abril - 2 de mayo de 2026.

## Outputs

- Project OS creado.
- Brief validado.
- Carpeta de referencias inicial.
- Tablero Notion definido o listo para crear.
- Primer moodboard visual.
- Primer cast bible.
- Primer mini animatic de 20-30 segundos o prueba de 3 tomas críticas.

## Tareas

| Tarea | Dueño | Estado |
|---|---|---|
| Confirmar dueños y aprobadores. | Producer | Pendiente |
| Crear tablero en Notion. | PM | Pendiente |
| Crear archivo Figma o localizar archivo maestro. | Design | Pendiente |
| Cargar 10 referencias visuales clave. | Creative | Pendiente |
| Diseñar UI T1 ficticia para tomas 03, 07, 09 y 11. | Design | Pendiente |
| Probar scratch VO con timing. | Audio / Creative | Pendiente |
| Generar stills de tomas 01, 02, 03, 11 y 12. | Prompt engineer | Pendiente |
| Revisar riesgos legales de primeras pruebas. | Legal | Pendiente |

## Criterio de éxito

Al final de Sprint 0 debemos poder decidir si la ruta visual es viable, qué tomas son las más riesgosas y qué personas/herramientas faltan para llegar al 26 de mayo.
